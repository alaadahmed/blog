About me

I'm an argentinian Optician with lots of interest in software development and the Elixir programming language in particular. I'm self taught, have good adaptability, and I'm always keen to learn more about technology, functional programming and computer science.

I volunteer as a moderator of the Elixir Discord server, where we provide assistance to other members of the Elixir community, and I'm the author of the Sourceror library for AST-based Elixir source code manipulation.

Open Source work

Sourceror
https://github.com/doorgan/sourceror
Sourceror is a toolkit to work with the Elixir source code, the first library in the Elixir ecosystem that empowers Elixirists and tool authors to build refactoring tools and migration scripts, similar to what jscodeshift has to offer for the JavaScript world.
Contributions to the Elixir language
I contributed a set of Pull Requests to expose functionality from the Elixir formatter to the users. In particular, the PR #10990 exposes the comments representation, and the PR #10988 implements a normalizer that is able to translate the regular elixir Abstract Syntax Tree into the extended, annotated AST used by the Elixir formatter.
YarnParser
https://github.com/doorgan/yarn_parser
A library to parse Yarn lockfiles, built with the NimbleParsec parser combinator library and used for a project in Georgia Tech Research Institute.
The Embers Social Network project
https://github.com/EmbersPlatform/embers
Embers was an exploratory project to build a social network platform with Elixir and the Phoenix framework. It used Phoenix Channels for a private chat and real time events, and Presence for an asymmetric list of online friends.
